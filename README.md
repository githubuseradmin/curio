# Curio — движок любопытства

Личный «движок любопытства»: короткие экспресс-карточки по многим темам — читаешь что-то новое каждый день, скипаешь известное, проваливаешься глубже, когда затянуло.

**🔗 Живая версия:** https://githubuseradmin.github.io/curio/

## Что это

Лента экспресс-карточек по 10 сферам: математика/вероятность, IT под капотом, кибербезопасность, ИИ изнутри, авиация, право, бизнес, экономика, английский, творчество. Каждая карточка — 4–10 минут: «интересная новость + объяснение под капотом», человеческим языком, с ссылками-«норами» вглубь и объяснением незнакомых слов по клику. Факты — со ссылками на источники.

## Возможности

- **Темп:** 1 / 3 / ∞ карточек за подход; видно, сколько минут в подборке.
- **Скип по-разному:** «Знаю» и «Не моё» — лента подстраивается под интересы.
- **Провалиться глубже:** «нора» из связанных карточек с навигацией назад (эффект Википедии).
- **🔎 Поиск** по всем темам · **🎲 Удиви меня** · **★ закладки** · стрик и счётчик изученного.
- **Термины** и «Словарь» — объяснение по клику.
- **Ссылка на карточку** (`#id` в URL) · **горячие клавиши** (`/`, `R`, `←`, `Esc`, `?`).
- Тёмная тема, адаптивно под телефон и ПК.

## Как открыть

- **Онлайн:** ссылка выше — работает и с телефона.
- **На телефон «как приложение»:** открой сайт в браузере → меню → «Добавить на главный экран». Откроется в полноэкранном виде со своей иконкой.
- **Локально:** открой `index.html`, либо подними статику: `python -m http.server 8801`.

## Устройство

Чистый статический сайт, ноль зависимостей и ноль бэкенда:

| Файл | Что |
|---|---|
| `index.html` | Всё приложение — UI и логика (ванильный JS). |
| `cards.js` | Контент: `window.CARDS = [...]`. |
| `manifest.webmanifest`, `icons/` | PWA: установка на телефон, иконки. |

Состояние (изученное, закладки, стрик) — в `localStorage` браузера.

## Статус

Личный прототип, для себя. Контент образовательный и местами упрощён ради ясности.
